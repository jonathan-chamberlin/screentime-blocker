Enable App Detection

This lets Brainrot Blocker detect when you're using desktop apps (like VS Code or Word) so your work timer runs even when the browser isn't in focus.

Installation Steps

  • 1 Find the native-host folder inside the extension files you downloaded If you downloaded from GitHub, paste %USERPROFILE%\Downloads into File Explorer's address bar to jump straight to your Downloads folder, then open the unzipped extension folder and look for native-host inside it
  • 2 Double-click install.bat
  • 3 When prompted, paste your Chrome extension ID Go to chrome://extensions, turn on 'Developer mode' in the top-right corner, and copy the ID shown under the extension name
  • 4 Restart Chrome completely (close all windows)
  • 5 Open Brainrot Blocker's Settings page â€” the 'Application detection is unavailable' warning should be gone

Troubleshooting

"Permission denied"
Try right-clicking install.bat and selecting "Run as administrator".
"Extension ID mismatch"
Make sure you copied the full ID from chrome://extensions. It's a 32-character string of lowercase letters, shown below the extension name when Developer mode is on.
"Still seeing warning"
Try restarting Chrome completely by closing all Chrome windows, not just the tab.
